# 软工3 文档仓库

## 课程要求

- 6 人一组，开源领域的软件工程实践（12k 代码）
- Agent 应用及评估系统搭建
- 三次迭代，每一次迭代都有不同的目标
  - 迭代 1：学习开源—阅读开源项目（Ragas 评估框架），完成基础评估实验
  - 迭代 2：基础搭建—初步搭建 Agent 应用与 Agent 评估平台
  - 迭代 3：深入探索—Agent 产品化与深度优化

## 给分

- 系统各阶段迭代产物（60％）
  - 迭代 1 → 迭代 2 → 迭代 3
- 软件工程过程产物（30％）
  - 提交文档及其他，个人／团队全部记录在 gitlab 上（个人工作评估根据个人对产物的贡献）
  - 如有异常，请及时邮件说明（房春荣 <fangchunrong@nju.edu.cn>）
- 其他过程（10％）
