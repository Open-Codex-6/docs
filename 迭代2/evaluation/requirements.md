# 需求

## 登陆/注册

- [ ] 注册：输入用户名和密码进行注册，注册成功后返回用户 id，失败时提示创建失败。
- [ ] 登录：输入用户名和密码登录，成功后返回 token 供后续使用，失败时提示用户不存在或密码错误。
- [ ] 更新用户信息：登录后携带 token 提交新的用户名和密码进行更新，成功时提示更新完成；若 token 无效、过期或与用户名不匹配则提示重新登录或校验失败。
- [ ] 获取所有 Agent 记录列表：支持分页、排序和筛选，每条记录包含：记录ID、发起用户、创建时间、综合评估分数、EPR是否通过、C‑LPR值、Ragas指标等
    - 筛选条件：时间范围、用户名称、综合分数区间、EPR状态、C-LPR值、Ragas指标等，
        
        默认以时间作为筛选条件。
    - 排序：按筛选条件升/降序，
    
        默认为降序。
- [ ] 获取某一次 Agent 记录的详细信息：记录的详细信息包含：发起用户、用户发问、Agent回答与执行、创建时间、各项评估指标及分值、综合评估分数，以及失败原因明细（如EPR失败项列表、LPR未满足约束列表）。
- [ ] 获取评估指标元数据：返回所有指标的名称、描述、取值范围、好坏方向、分类（硬约束/软偏好）。
- [ ] 获取某个指标的历史趋势数据（用于展示指标曲线）：根据给定的指标名称和近x次（或时间范围），返回每条记录的时间戳、记录ID、指标值、以及该记录是否通过EPR。
