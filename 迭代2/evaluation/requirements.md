# 需求

## 登陆/注册

- [ ] 注册：输入用户名和密码进行注册，注册成功后返回用户 id，失败时提示创建失败。
- [ ] 登录：输入用户名和密码登录，成功后返回 token 供后续使用，失败时提示用户不存在或密码错误。
- [ ] 更新用户信息：登录后携带 token 提交新的用户名和密码进行更新，成功时提示更新完成；若 token 无效、过期或与用户名不匹配则提示重新登录或校验失败。

## Agent评估计算

- [ ] 评估计算：提供评估触发接口，入参包含足以确定一次 Agent 会话的数据（完整内容或标识符）。评估平台根据规则计算指标并持久化结果。

## Agent评估数据

### 前端

- [ ] 展示所有评测记录：列表支持筛选，排序，分页
- [ ] 仪表盘功能，类似于数据大屏，能看所有评估指标的状态、曲线、变化
- [ ] 导出报表/报告

### 后端

- [ ] 获取所有评测记录列表：每条记录包含：记录 ID、发起用户、创建时间、综合评估分数、EPR 状态、C‑LPR 值、Ragas 指标、Tag 等评测记录基础信息
    - 支持筛选：
      - 筛选条件：可选，时间范围、用户名称、综合分数区间、EPR 是否通过、C-LPR 值区间、Ragas 指标区间、Tag 等，**默认无筛选**。
    - 支持排序、分页：
      - 排序条件：可选，时间、综合分数、EPR 状态、C-LPR 值、Ragas 指标等，**默认以时间作为排序条件**。
      - 排序：可选，按排序条件升/降序，**默认为降序**。
      - 分页参数：
        - 页号：必选，整数，从1开始
        - 分页数目：可选，整数，**分页数目默认为10条**
- [ ] 获取某一次评测记录的详细信息：记录的详细信息包含：发起用户、用户发问、Agent 回答与执行、创建时间、各项评估指标及分值、综合评估分数，以及失败原因明细（如 EPR 失败项列表、LPR 未满足约束列表）。
- [ ] 获取评估指标元数据：返回所有指标的名称，以及其对应的描述、取值范围、好坏方向、分类（硬约束/软偏好）等属性。
- [ ] 获取某个指标的历史趋势数据（用于展示指标曲线）：根据给定的指标名称和近 x 次（或时间范围），返回每条记录的时间戳、记录 ID、指标值、以及该记录是否通过 EPR。
